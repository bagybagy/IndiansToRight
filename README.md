# インド人を右に！

群衆を左右へ押し寄せ、満員バスの重心をカーブ内側へ寄せて突破する Three.js プロトタイプです。

## 起動

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

ブラウザで `http://127.0.0.1:5173/` を開きます。

## 操作

- `←` / `A`: 乗客を車内左へ押す
- `→` / `D`: 乗客を車内右へ押す
- `Space`: カーブ判定
- `R`: リセット

## 実装方針

- 画面左右を車内左右として扱う
- 視点はTPS寄りのやや上から俯瞰。道路は奥へ進み、次カーブ方向に曲がる
- 操作ループは「直線で重心準備 → カーブ突入時だけ遠心力判定」
- 寄せ不足は外側へ横転しそうになり、寄せすぎは内側へ片輪走行しそうになる
- 直線準備中でも寄せすぎると横転寸前まで倒れ込み、乗客が降車する
- 横転は揺れ演出ではなく、重心とカーブ外力から決まる倒れ角が限界を超えた状態として扱う
- 車体の大きな揺れや片輪走行時は、乗客の塊も遅れて左右に振られる
- 乗客は物理シミュレーションではなく、位置・エリア・重みを持つローポリ記号として扱う
- 重心は各乗客の実位置と重みから計算する
- 人数増加に応じて、車内、窓、屋根、側面へ溢れる
- 満員域ではダンス状態になり、速度と画面演出が上がる
